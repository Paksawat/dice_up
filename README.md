# Flutter Demo

A simple Flutter demo application with a counter.

## Getting Started

### Prerequisites

- Flutter SDK installed (version 3.0.0 or higher)
- Dart SDK (comes with Flutter)

### Installation

1. Install dependencies:
   ```bash
   flutter pub get
   ```

2. Run the app:
   ```bash
   flutter run
   ```

### Features

- Simple counter app
- Material Design 3
- Responsive UI

## Project Structure

```
lib/
  main.dart       # Main application entry point
pubspec.yaml      # Project dependencies and configuration
```
